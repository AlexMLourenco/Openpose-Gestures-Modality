1. Install Python
2. Install Python requirements (pip install -r requirements.txt)
3. Change OPEN_POSE_DIR variable on run_openpose_IM.py to the root of the open pose directory (e.g. C:\\OPENPOSE_CPU_ONLY\\openpose-1.5.1-binaries-win64-only_cpu-python-flir-3d\\openpose). Use double back-slashes
4. Run run_openpose_IM.py. Pictures from videos to analyze should be on folder videos and inside a folder.
If any error, see logs on logs folder.
